# 计算机探索（不作弊版）

本目录用于继续推进“介质动力学计算”实验，严格遵守：

- 禁止 `sqrt`
- 禁止牛顿迭代
- 禁止后置兜底提纯（如二分扫描修正结果）
- 允许失败，保留真实残差

## 当前目标

从“单点验证”升级到“方法学验证”：

1. 任务泛化：不仅算 `sqrt(2)`，还尝试 `sqrt(3)`、`1/x` 等
2. 相图稳定性：给出收敛/振荡/漂移边界
3. 可编程性：将映射函数参数化，形成最小“计算指令”

## 目录说明

- `run_exploration.ps1`：一键运行脚本（conda 激活后执行）
- `experiment_log.md`：手工记录每轮实验结论
- `figures/`：图像输出目录

## 建议工作流

1. 先跑基础相图，确认当前映射的行为类型
2. 只改一项变量（映射/参数/初值）再复跑
3. 每次把“是否更接近目标 + 是否更稳定”写入日志

## 审计准则

- 结果若不收敛，不隐藏、不兜底
- 只报告“模型真实给出的终态”
- 所有阈值都写明，避免后验调参误导
