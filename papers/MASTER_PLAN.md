# 论文总控计划（统一标准，双线独立）

本项目采用“双论文并行、证据链独立”的工作方式：

- Bell 线：`papers/bell-audit-paper/`
- GHZ 线：`papers/ghz-threebody-paper/`

## 统一质量门槛（两篇都必须满足）

- [ ] 可复现：固定依赖、固定命令、固定随机种子策略
- [ ] 可审计：原始 JSON、汇总表、图表、脚本可追溯
- [ ] 可统计：多 seed、置信区间、稳健性分析
- [ ] 可边界化：明确“方法结论”与“物理本体结论”的边界
- [ ] 可投稿：中英文稿件结构完整，图表编号一致

## 统一交付物（每篇论文）

1. `TODO.md`：严谨任务清单（逐项打勾）
2. `draft.zh.md`：中文稿
3. `draft.en.md`：英文稿
4. `figures/`：图表与图注
5. `tables/`：主表与附表
6. `appendix/`：补充实验与推导

## 执行原则

- 先完成方法与审计协议，再写结果，不先下结论。
- 每次新增结论必须对应“数据证据 + 图表 + 命令”。
- 不删除负结果，负结果是审计可信度的一部分。
