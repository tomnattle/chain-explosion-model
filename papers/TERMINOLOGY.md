# 中英术语统一表（Bell + GHZ）

## 核心术语

| 中文 | English | 说明 |
|---|---|---|
| 分母审计 | denominator audit | 对归一化分母与样本包含规则的核查 |
| 协议簿记 | protocol bookkeeping | 指协议定义导致的统计口径差异 |
| 后选择 | post-selection | 对事件样本的条件筛选 |
| 一致性门槛 | gate criteria | 工程门槛与论文门槛的统一称呼 |
| 工程门槛 | engineering gate | 可计算性/样本量等执行门槛 |
| 论文门槛 | thesis gate | 论文主张所需的判据门槛 |
| 审计员效应 | auditor effect | 规则选择在解释前已显著塑造统计结果的现象 |
| 严格协议 | strict protocol | 更严格配对窗口或更严格规则 |
| 标准协议 | standard protocol | 标准或较宽窗口协议 |
| 两阶段搜索 | two-stage search | 粗搜 + 细搜 |
| 粗搜 | coarse search | 第一阶段网格扫描 |
| 细搜 | fine search | Top-k 邻域高分辨率扫描 |
| 稳健性 | robustness | 多 seed / bootstrap / 扰动一致性 |
| 交易关系 | trade-off | 相关性与样本率之间的约束关系 |
| 方法级结论 | method-level conclusion | 仅限审计协议内可确认结论 |
| 本体级结论 | ontology-level conclusion | 对基础物理叙事的外推结论 |

## 书写规则

1. 中文稿首次出现英文术语时，用“中文（English）”格式。
2. 英文稿避免混用近义词，统一使用本表英文列。
3. “工程门槛/论文门槛”在英文稿统一为 `engineering gate` / `thesis gate`。
4. 不使用“证明了新物理”等超边界措辞，统一用 `method-level conclusion`。
