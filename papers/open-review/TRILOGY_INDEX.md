# Audit Trilogy Index

## Overview

This index summarizes the three published preprints in the Bell/GHZ audit trilogy and provides direct DOI access for expert review and reproducibility checks.

## Paper I — Bell Audit

- Title: `Denominator-Audit Robustness Analysis for Bell/CHSH Statistics: A Reconciliation Study of Strict vs Standard Protocols`
- DOI: `https://doi.org/10.5281/zenodo.19763242`
- Core finding: On the same NIST event stream, pairing-window/protocol changes shift CHSH from `S_strict=2.336276` to `S_standard=2.839387` (`Delta=+0.503111`).
- Role in trilogy: establishes protocol sensitivity as an auditable first-class variable.

## Paper II — GHZ Audit (Final Edition)

- Title: `The Geometric Origin of GHZ Violation: A Local Realist Deconstruction via Phase-Loop Interference and Threshold Triggering`
- DOI: `https://doi.org/10.5281/zenodo.19769427`
- Core finding: Demonstrates exact replication of `F=4.000000` using a local phase-loop model with a non-linear threshold trigger.
- Role in trilogy: Provides the definitive local realist ontological solution for the GHZ violation.

## Paper III — Trilogy Synthesis

- Title: `Analysis Rule Sensitivity in Bell and GHZ Experiments: A Reproducible Audit Framework`
- DOI: `https://doi.org/10.5281/zenodo.19764505`
- Core finding: integrates Bell and GHZ evidence into a ledger-style framework with explicit claim boundaries and claim-to-artifact mapping.
- Role in trilogy: defines the reusable audit standard for future high-correlation reporting.
