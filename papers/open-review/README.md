## Suggested Assets for Outreach

- Core message in 5-8 lines (problem, method, key outcome, limitation).
- 1-2 key figures:
  - ../../artifacts/ghz_threshold_experiment/ghz_threshold_F_vs_T.png
  - ../../artifacts/ghz_threshold_experiment/ghz_threshold_mechanism_heatmap.png
- Minimal reproduction command(s) from repository root.

## Recommended Process

1. Publish/update preprint + keep GitHub public.
2. Send short technical feedback email (not a full paper dump).
3. Ask one concrete question per expert.
4. Follow up once after 10-14 days if no response.
