## 1. **致命错误：external_clock_bin 里做了「一对一最佳匹配」，直接引入后选漏洞（Post-selection）**
### 代码位置
```python

    ...
    for _, a in aa:
        best_j = -1
        best_dt = None
        best_b = None

            dt = abs(b.t - a.t)
            if best_dt is None or dt < best_dt:
                best_dt = dt
```

### 问题
**external clock bin = 时间片桶**
标准 NIST 做法是：**同桶内全部计数，不做最佳匹配**。
你代码里：
- 对每个 A，**挑时间差最小的 B**
- 标记 used_b，不许重复使用

这叫 **greedy one-to-one nearest matching**
→ **人为挑选最相关的事件对**
→ 强行抬高相关性
→ 直接制造高 S 值

### 结论
你现在的 `external_clock_bin` **不是 NIST 官方算法**，
是你自己加了**强筛选配对**，
所以你跑出 2.82，**是你代码造出来的，不是实验真实结果**。

## 4. **严重错误：cont_norm 归一化在 Bell 测试中是不合法操作**
### 代码位置
```python
num = float(np.mean(xa[m] * xb[m]))
den = float(np.sqrt(max(1e-12, float(np.mean(xa[m] * xa[m])) * float(np.mean(xb[m] * xb[m])))))

```

### 问题
**Bell-CHSH 测试严禁对每个设置组单独归一化！**
标准定义：
$$ E = \frac{N_{++}+N_{--}-N_{+-}-N_{-+}}{N_{++}+N_{--}+N_{+-}+N_{-+}} $$
你用：
$$ E = \frac{\langle xy\rangle}{\sqrt{\langle x^2\rangle\langle y^2\rangle}} $$
这是**皮尔逊相关系数**，不是 CHSH！

### 结论
你用**相关系数冒充 CHSH**，跑出 2.33～2.82，
**在量子光学里属于无效统计量**。
